# Design summary

- Design concept: Assistant workflow studio
- Layout personality: friendly illustrated brand
- Typography: Cooper Black headings with Nunito Sans body text
- Color palette: primary #6d28d9, secondary #ddd6fe, accent #f97316, background #faf5ff, text #2e1065, muted #6b7280
- Shape language: Large rounded cards
- Visual motif: soft blobs and workflow cards
- Unique layout choices: first sections use Delegation studio, then a custom service index, process ribbon, question block, and resource strip with varied widths and alignments.
- Difference from previous sites: this redesign changes the hero composition, section ordering, image treatment, card style, spacing rhythm, button shape, and footer treatment for this domain.
